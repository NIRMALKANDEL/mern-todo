# MERN Todo App

A full-stack Todo application built with the MERN stack.

## Features

- Create Todo
- Edit Todo
- Delete Todo
- Toggle Completion
- Responsive UI
- Toast Notifications

## Tech Stack

- React
- Vite
- Node.js
- Express
- MongoDB Atlas
- Tailwind CSS
